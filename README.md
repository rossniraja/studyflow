# 📚 StudyFlow — Smart Study Tracker
 
A clean, dark-themed productivity web app designed to help students track study sessions, manage tasks, and analyze their learning patterns — all in one place.
 
---
 
## 🔗 Live Demo
 
[View Live →](https://studyflow-studytracker.netlify.app/) 
 
---
 
## ✨ Features
 
- **Dashboard** — Overview of daily and weekly study hours, task progress, focus score, and daily goals
- **Focus Timer** — Pomodoro-based focus timer with Short Break and Long Break modes, session subject tracking, and session history
- **Task Manager** — Add, view, and manage study tasks with subject tagging and priority levels
- **Subjects** — Track time spent per subject across the month
- **Analytics** — Study heatmap, peak focus hours visualization, and monthly stats
---
 
## 🛠️ Built With
 
- **HTML5** — Semantic structure and multi-page layout
- **CSS3** — Custom properties (CSS variables), Flexbox, Grid, animations
- **Vanilla JavaScript** — DOM manipulation, event handling, dynamic rendering, Pomodoro timer logic
> No frameworks. No libraries. Pure frontend.
